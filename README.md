# Mangalam HDPE Pipes Website

## Overview
This is a responsive website for Mangalam HDPE Pipes, showcasing their premium HDPE pipes and coils for modern infrastructure. The site includes product information, technical specifications, resources, and contact forms.

## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Hero Section**: Product showcase with gallery, pricing, and call-to-action buttons
- **Technical Specifications**: Detailed specs table with download options
- **Resources**: Downloadable PDFs for manuals and specifications
- **Modals**: Popup forms for requesting quotes and downloading datasheets
- **Sticky Header**: Appears on scroll for easy navigation
- **Interactive Elements**: FAQ accordion, image zoom, carousels
- **Contact Form**: Integrated contact section with form submission
